# Proposed Codex Modules

This directory holds **proposed**, **pre-derivation**, **non-canonical** module proposals — synthesis output from AI sessions that introduces new equations, symbols, or modules that have not yet been:

- derived from the existing codex spine (G₂ representation theory, torsion decomposition, PSL(2,7) coset structure, the 6/7 contraction);
- subjected to the modeling-choice-stack discipline applied to Paper-grade material (e.g., `outbox/paper7/` §4.1, §4.2.2, §4.3, §4.4);
- run through a Model Council adversarial review pass.

**Files in this directory are not part of the canonical codex.** They are working proposals preserved for future development. Do **not** merge their equations or symbols into the canonical CSVs (`codex/pci_equations_v63_cleaned.csv`, `codex/pci_symbols_v47_cleaned.csv`, or the partition-index files) without:

1. Deriving each proposed quantity from existing canonical structure (or explicitly labelling it as a free-parameter modeling layer that does not claim derivation), and
2. A Council-grade integrity pass on the proposed addition.

This directory is the equivalent of a research notebook: ideas live here until they earn their way into the spine.
