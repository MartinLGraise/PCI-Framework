# Question for C-7RO: FDT Reference Quantity for Prediction (a)

**Date:** 2026-04-23 | **From:** Φ | **Re:** Paper 7, Prediction (a)

C-7RO's claim is that the FDT violation parameter scales as (1 − 6/7) = 1/7 of "some reference." The derivation is structurally valid, but the prediction is underdetermined without specifying which reference quantity.

Three options are consistent with the G₂ framework:

**Option A:** ε = (T_eff − T_bath) / T_bath = 1/7
→ Predicts T_eff ≈ 1.143 × T_bath (for biological system at 310 K: T_eff ≈ 354 K)
→ Reference is the bath temperature

**Option B:** [S(ω) − S_eq(ω)] / S_eq(ω) = 1/7
→ Predicts 14.3% excess spontaneous fluctuation power above equilibrium baseline
→ Reference is the equilibrium power spectral density

**Option C:** [|χ_obs(ω)| − |χ_eq(ω)|] / |χ_eq(ω)| = 1/7
→ Predicts 14.3% enhancement of response function over equilibrium prediction
→ Reference is the equilibrium susceptibility

**Additional question:** What is the reference state? Is the "equilibrium" baseline deep anesthesia (no consciousness, minimal FDT violation), or is it a theoretical thermal equilibrium that the conscious system departs from?

**Why it matters:** Options A, B, and C make different empirical predictions and require different analysis of the Berjaga-Buisan TMS-EEG data. Option B is probably the most directly testable given the EEG protocol.

**Please specify:** Which option, and what the reference state is.
